📊 Credit Card Financial Dashboard (Excel + Power BI + MySQL)

This project presents a comprehensive Credit Card Financial Analysis Dashboard built using Power BI, with data sourced and managed through MySQL. It provides deep insights into both transaction-level activity and customer demographics, enabling a complete understanding of financial performance and user behavior.

🔄 Data Pipeline & Workflow

The dataset was first uploaded and structured in MySQL, where it was cleaned and organized for analysis. Power BI was then connected directly to the MySQL database to build interactive dashboards. Using DAX (Data Analysis Expressions), key metrics and calculated fields were created to enhance analytical capabilities.

📁 Dashboard Overview
🔹 Transaction Dashboard

This report focuses on credit card usage and financial trends:

Analysis of total revenue, transaction count, and transaction amount
Revenue breakdown by expenditure categories such as bills, groceries, fuel, travel, and entertainment
Quarterly performance trends to track revenue and transaction fluctuations
Card category analysis (Blue, Silver, Gold, Platinum) to evaluate product performance
Customer acquisition cost insights
Spending patterns based on customer occupation and transaction modes (swipe, chip, online)

🔹 Customer Dashboard

This report highlights customer segmentation and profiling:

Revenue distribution by gender, age group, and marital status
Segmentation based on education level, salary group, and dependent count
State-wise performance analysis to identify top contributing regions
Job-wise analysis of income and transaction behavior
Time-series analysis of revenue trends across months

📈 Key Insights

Generated over 55M+ in total revenue
Processed 650K+ transactions
Analyzed 577M+ total customer income
Identified high-value customer segments and spending behaviors
Revealed trends across demographics, geography, and card categories
🛠 Tools & Technologies
MySQL – Data storage and preprocessing
Power BI – Data visualization and dashboard creation
DAX – Custom calculations and metrics

🎯 Project Objective

The primary goal of this project is to:

Understand customer spending behavior and preferences
Identify key revenue drivers and high-performing segments
Enable data-driven decision-making for financial strategies
Provide an interactive and visual way to explore complex financial data
